BrazmockArcheryMod

Place the BrazmockArcheryMod folder into your mods folder which is located at:

Program Files (x86)/Steam/steamapps/common/7 Days to Die

If you dont have a mods folder, you can create one my right clicking and clicking on 'new' and
then folder and name it exactly Mods (case sensitive) then place the mod folder in that folder.
